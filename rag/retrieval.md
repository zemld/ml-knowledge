# Типы ретриверов в RAG

> Ретривер определяет, как именно система ищет релевантные чанки — по смыслу (dense), по ключевым словам (sparse) или совмещая оба подхода (hybrid).

## Dense-ретривер (семантический поиск)

Использует **эмбеддинг-векторы** для поиска по смыслу. Запрос и документы преобразуются в векторы одной моделью; ближайшие по косинусному расстоянию чанки считаются релевантными.

**Плюсы:** находит чанки, сформулированные иначе, чем запрос (синонимы, перефразировки).  
**Минусы:** может пропустить точные термины — имена продуктов, версии, идентификаторы.

Recall@10: около **78%**.

## Sparse-ретривер (поиск по ключевым словам)

Использует алгоритмы типа **BM25** или **TF-IDF** — классический keyword-поиск. Ищет документы, буквально содержащие слова из запроса.

**Плюсы:** надёжен для технической терминологии, артикулов, имён собственных, номеров версий.  
**Минусы:** не понимает смысла — «автомобиль» и «машина» это разные слова.

Recall@10: около **65%**.

## Гибридный поиск (Hybrid Search)

Запускает оба ретривера параллельно и объединяет результаты. Стандартный метод слияния — **Reciprocal Rank Fusion (RRF)**:

$$\text{RRF}(d) = \sum_{r \in R} \frac{1}{k + r(d)}$$

где $r(d)$ — позиция документа $d$ в ранжированном списке ретривера $r$, $k$ — константа (обычно 60).

Recall@10 гибридного поиска: около **91%** — на 17% выше, чем у dense-only.

## Reranker (опционально)

После получения Top-K кандидатов от гибридного ретривера можно применить **cross-encoder reranker** (например, Cohere Rerank) — он перечитывает пары (запрос, чанк) и переупорядочивает результаты с более высокой точностью, чем bi-encoder.

Cross-encoder медленнее и не подходит для поиска по всему индексу, но отлично работает для перепроверки небольшого числа кандидатов.

## Сравнение

| Подход | Recall@10 | Хорошо для | Слабо для |
|---|---|---|---|
| Dense | ~78% | Семантическое сходство | Точные термины |
| Sparse (BM25) | ~65% | Имена, версии, технические термины | Перефразировки |
| **Hybrid** | **~91%** | Всё вышеперечисленное | — |
| + Reranker | выше | Финальная точность Top-K | Скорость |
