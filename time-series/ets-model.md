# ETS-модели (экспоненциальное сглаживание)

> ETS — унифицированное семейство моделей прогнозирования, где каждая модель задаётся тремя компонентами: Error (ошибка), Trend (тренд), Seasonality (сезонность); охватывает всё от простого сглаживания до метода Холта-Уинтерса.

## Что такое ETS

**ETS** расшифровывается как **E**rror – **T**rend – **S**easonality. Каждая из трёх букв принимает значение:

- **N** (None) — компонент отсутствует
- **A** (Additive) — компонент аддитивный
- **M** (Multiplicative) — компонент мультипликативный

Запись `ETS(A, A, N)` означает: аддитивные ошибки, аддитивный тренд, без сезонности. Это задаёт конкретную модель из семейства.

Исторически семейство строилось постепенно: Браун (1956) предложил простое экспоненциальное сглаживание, Холт (1957) добавил тренд, Уинтерс (1960) — сезонность. Хиндман и коллеги (2002) объединили всё в единый state space фреймворк ETS.

## Простое экспоненциальное сглаживание — ETS(A,N,N)

Прогноз на следующий шаг — взвешенное среднее текущего наблюдения и предыдущего прогноза:

$$\hat{y}_{t+1} = \alpha\, y_t + (1 - \alpha)\, \hat{y}_t$$

где $\alpha \in (0, 1]$ — **параметр сглаживания**: чем $\alpha$ ближе к 1, тем сильнее модель опирается на последнее наблюдение; чем ближе к 0 — тем «инертнее».

Развернув рекурсию, видно, что веса убывают экспоненциально: наблюдение в момент $t-k$ имеет вес $\alpha(1-\alpha)^k$.

Работает только если в ряду нет тренда и сезонности.

## Основные модели ETS

### ETS(A,N,N) — простое сглаживание

Только уровень $\ell_t$, нет тренда и сезонности. Прогноз на любой горизонт — константа: $\hat{y}_{t+h|t} = \ell_t$.

### ETS(A,A,N) — метод Холта

Добавляется тренд $b_t$:

$$\ell_t = \alpha\, y_t + (1 - \alpha)(\ell_{t-1} + b_{t-1})$$
$$b_t = \beta(\ell_t - \ell_{t-1}) + (1 - \beta)\, b_{t-1}$$

Прогноз: $\hat{y}_{t+h|t} = \ell_t + h\, b_t$ — линейная экстраполяция тренда.

### ETS(A,A,A) — метод Холта-Уинтерса (аддитивный)

Добавляется сезонный компонент $s_t$ с периодом $m$. State space уравнения:

$$y_t = \ell_{t-1} + b_{t-1} + s_{t-m} + \varepsilon_t$$
$$\ell_t = \ell_{t-1} + b_{t-1} + \alpha\, \varepsilon_t$$
$$b_t = b_{t-1} + \beta\, \varepsilon_t$$
$$s_t = s_{t-m} + \gamma\, \varepsilon_t$$

Прогноз на $h$ шагов:

$$\hat{y}_{t+h|t} = \ell_t + h\, b_t + s_{t+h-m(k+1)}$$

где $k = \lfloor (h-1)/m \rfloor$.

### ETS(M,M,M) — Холт-Уинтерс мультипликативный

Используется, когда амплитуда сезонности растёт вместе с уровнем ряда (см. [[time-series/decomposition|Декомпозиция]]). Компоненты перемножаются вместо сложения.

## Как ETS строит прогноз на H шагов

ETS — **рекуррентная** модель: прогноз на шаг $h$ строится через состояния $(\ell_t, b_t, s_t)$, обновлённые по последнему наблюдению. Состояния распространяются вперёд детерминированно:

1. Оценить начальные состояния и параметры $(\alpha, \beta, \gamma)$ методом максимального правдоподобия.
2. Обновить состояния на каждом историческом шаге.
3. Для горизонта $h$ — подставить состояния на момент $T$ в аналитическую формулу прогноза.

State space формулировка также даёт **доверительные интервалы** прогноза через распределение ошибок.

## Выбор модели

Оптимальную модель ETS можно выбрать автоматически по минимуму информационного критерия AICc (скорректированный AIC), перебрав все разумные комбинации типов компонент. Именно так работает функция `auto_arima` в `statsforecast` и `ets()` в R.

**Предупреждение:** комбинация аддитивных ошибок с мультипликативной сезонностью `ETS(A,*,M)` может быть численно нестабильной — предпочтительнее `ETS(M,*,M)`.
