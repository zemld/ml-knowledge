# Декомпозиция временного ряда

> Разложение ряда на тренд, сезонность и остаток позволяет понять структуру данных и упрощает прогнозирование; классический метод использует скользящее среднее, а STL — локальную регрессию и устойчив к выбросам.

## Компоненты временного ряда

Любой временной ряд можно представить как комбинацию трёх составляющих:

- **Тренд ($T_t$)** — долгосрочное направление ряда: рост, спад или горизонталь. Например, ежегодный рост выручки компании.
- **Сезонность ($S_t$)** — повторяющиеся паттерны фиксированного периода: пики продаж в декабре, провалы трафика по выходным.
- **Остаток ($R_t$)** — случайный шум, который не объясняется трендом и сезонностью.

## Аддитивная и мультипликативная модели

Различаются по тому, как компоненты соединяются:

**Аддитивная модель:**

$$y_t = T_t + S_t + R_t$$

Применяется, когда амплитуда сезонных колебаний остаётся примерно постоянной независимо от уровня ряда. Например, температура воздуха колеблется на ±10°C в любой год.

**Мультипликативная модель:**

$$y_t = T_t \times S_t \times R_t$$

Применяется, когда амплитуда сезонности масштабируется вместе с уровнем ряда. Например, у быстрорастущего интернет-магазина новогодний пик в абсолютных числах год от года увеличивается пропорционально выручке.

**Переход между моделями:** если нужна аддитивная декомпозиция, но ряд мультипликативен, достаточно взять $\log(y_t)$ и работать с логарифмированным рядом.

## Классическая декомпозиция

Оценивает тренд через **центрированное скользящее среднее** периода $m$ (длина сезонного цикла), затем извлекает сезонность усреднением по соответствующим позициям цикла, а остаток — вычитанием (или делением для мультипликативной модели).

**Ограничения:**
- Значения тренда недоступны на концах ряда (нет данных для центрирования).
- Сезонность считается абсолютно постоянной — паттерн не может эволюционировать.
- Чувствительна к выбросам.

## STL-декомпозиция

**STL** (Seasonal-Trend decomposition using LOESS) — современный стандарт. Предложен Cleveland et al. в 1990 году.

**LOESS** (Locally Estimated Scatterplot Smoothing) — метод локальной регрессии: вместо одного глобального тренда подбираются малые регрессии на перекрывающихся окнах, которые затем объединяются в плавную кривую.

**Как работает STL:** итеративный алгоритм с двумя петлями.
1. **Внутренняя петля:** оценивает сезонность через LOESS → вычитает её → оценивает тренд через LOESS на десезонированном ряду.
2. **Внешняя петля (опционально):** пересчитывает веса наблюдений, снижая влияние выбросов.

**Ключевые параметры:**
- `season` — ширина окна сглаживания сезонности (нечётное число, например 13 для месячных данных с годовой сезонностью).
- `trend` — ширина окна тренда (обычно ~150% от `season`, тоже нечётное).

**Преимущества STL перед классической декомпозицией:**

| Свойство | Классическая | STL |
|---|---|---|
| Сезонность может меняться | ✗ | ✓ |
| Устойчивость к выбросам | ✗ | ✓ |
| Работа с концами ряда | Пропуски | ✓ |
| Мультипликативная напрямую | ✓ | ✗ (нужен log) |
