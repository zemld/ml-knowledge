# Предобработка аудио

> Набор обязательных шагов преобразования сырого аудиосигнала в числовые признаки, пригодные для обучения нейронных сетей.

## Зачем нужна предобработка

Нейронные сети не работают с сырыми аудиофайлами напрямую: файлы отличаются по частоте дискретизации, громкости и длине, а временной сигнал плохо подходит для выявления частотных паттернов. Предобработка решает обе проблемы: сначала приводит сигнал к единому формату (временная область), затем переводит его в частотно-временное представление (спектральные признаки).

## Типичный пайплайн

```
Аудиофайл
  → Ресемплинг (единая частота дискретизации)
  → Нормализация амплитуды
  → Конвертация в моно
  → Удаление тишины
  → STFT (оконное преобразование Фурье)
  → Мел-спектрограмма / Log-mel / MFCC
  → Вход модели
```
