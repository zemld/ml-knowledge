# Нейросети

> Область машинного обучения, изучающая алгоритмы, вдохновлённые строением мозга: слои «нейронов» учатся находить паттерны в данных и решать задачи, которые трудно описать явными правилами.

## Что такое нейросеть

Нейронная сеть — это математическая функция, составленная из слоёв простых операций. Каждый «нейрон» получает числа на вход, умножает их на веса, складывает и пропускает через нелинейность (например, ReLU или sigmoid). Обучение — это подбор весов так, чтобы выходы сети совпадали с ожидаемыми ответами.

Нейросети применяются там, где входные данные слишком сложны для ручного написания правил: распознавание изображений, перевод текста, синтез речи, игра в го.
