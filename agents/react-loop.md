# Цикл ReAct: Thought → Action → Observation

> ReAct (Reasoning + Acting) — паттерн работы агента, при котором LLM чередует явные рассуждения и вызовы инструментов, используя наблюдения для коррекции следующего шага.

## Три фазы одного витка

### Thought (Мысль)

LLM генерирует текст-рассуждение: что известно на данный момент, что нужно сделать дальше и почему. Это внутренний монолог агента, невидимый конечному пользователю.

```
Thought: Мне нужно узнать текущий курс доллара.
         Использую инструмент поиска.
```

### Action (Действие)

На основе мысли агент выбирает инструмент и передаёт ему параметры. Выход модели маршрутизируется в реальный вызов API или функции.

```
Action: search("курс доллара к рублю сегодня")
```

### Observation (Наблюдение)

Результат вызова инструмента возвращается в контекст агента. **Наблюдение генерирует не LLM** — оно приходит из внешней системы и служит фактической точкой опоры, предотвращая галлюцинации.

```
Observation: "По данным ЦБ РФ на 25.05.2026: 1 USD = 83.41 RUB"
```

## Полный пример витка

```
Цель: Сколько стоит 500 долларов в рублях?

Thought: Нужен актуальный курс доллара.
Action: search("курс доллара ЦБ РФ 2026-05-25")
Observation: 1 USD = 83.41 RUB

Thought: Курс получен. Вычислю: 500 × 83.41 = 41 705.
Action: calculator("500 * 83.41")
Observation: 41705.0

Thought: Ответ готов.
Answer: 500 долларов = 41 705 рублей по курсу ЦБ на 25.05.2026.
```

## Как строится контекст между витками

После каждого витка вся тройка `Thought / Action / Observation` добавляется в контекст следующего вызова LLM. Модель «видит» всю историю рассуждений и может корректировать курс при неожиданных наблюдениях.

```
[System prompt + описания инструментов]
[Цель]
[Thought 1] [Action 1] [Observation 1]
[Thought 2] [Action 2] [Observation 2]
...
```

## Условие остановки

Без явного условия агент может зациклиться. Типичные механизмы завершения:
- LLM генерирует специальный токен/тег `Answer:` — сигнал финального ответа.
- Достигнут лимит итераций (жёсткий потолок числа витков).
- Выполнено условие задачи (проверяется внешне).
